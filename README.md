# NATA Today V1

Static Next.js/Vercel demo for NATA Today.

## Routes

- `/` — Homepage
- `/dealer-demo` — Dealer command dashboard
- `/candidate-demo` — Candidate readiness profile

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and import into Vercel, or upload the project directory to Vercel.

## Notes

This version intentionally has no Supabase dependency. It is built to prove the visual face and product direction first.
